Ensemble methods

Ensemble methods

Main Idea

​ Multiple models (aka “weak learners”) are trained to solve the same problem and combined to get better results.

How to combine?

  1. Train different models independently and take a average of all models - Bagging (such as random forest)
  2. Train a base model first. And we train next model to make up for the disadvantage of the previous model, which is adaptive. - Boosting

Boosting

The main idea of the boosting is to combine a set of models with low variance but high bias to reduce the bias and get a strong one.( The bagging do the different. ) Boosting mainly focused at reducing bias.

Boosting is a sequential methods, we fit the models iteratively and at each step we hope to train a new model which can improve our pervious model.

How we sequentially fit the model?

  1. Adaptive boosting updates the weights attached to each of the training dataset observations.
  2. Gradient boosting updates the value of these observations.(The loss function)

Gradient boosting

The main goal of the Gradient boosting is to find an approximation $\hat{F}(x)$ to a function $F(x)$ that minimizes the expected value of a specified loss function $L(y, F(x))$:

​${\hat {F}}={\underset {F}{\arg \min }},\mathbb {E}_{x,y}[L(y,F(x))]$

Our final model $\hat{F}(x)$ is a combination of many base model (aka weak learner), and it can be represented a weighted sum of these base model. It’s formula is as following:

​${\hat{F}}(x)=\sum_{i=1}^{M}\gamma_{i}h_{i}(x)+{\mbox{const}}$

Where $h_i(x)$ is the base model.

Review the Gradient descent

​$w_{i+1} = w_i - \lambda \frac{\partial\cal{L(w)}}{\partial w}$

In the Gradient boosting, we consider the $F(x)$ as the parameter, then we can update the our model as:

​$F_m(x) = F_{m-1}(x) - \gamma_m \frac{\partial L(y, F_{m-1}(x))}{F_{m-1}(x)}$

Since for an arbitrary loss function $L(x)$ , it is infeasible to solve such a optimization problem. We use a base learner $h(x)$ to fit negative gradient of each step, then we can update the model as follow:

​$h_m(x) \approx -\frac{\partial L(y, F_{m-1}(x))}{F_{m-1}(x)}$

​$F_m(x) = F_{m-1}(x) + \gamma_m h_m(x)$

Where the $\gamma_m$ is the step length.

At each iteration, we first fit a base model $h_m(x)$ to the negative gradient and we hope to set a proper step length $\gamma_m$ to reduce the loss function $L(y, F(x))$.

​$\gamma_{m}={\underset{\gamma}{\arg \min }}{\sum_{i=1}^{n}{L\left(y_{i},F_{m-1}(x_{i})+\gamma h_m(x)\right)}}$

However this is a heuristic method, for the complex loss function we can get the exact result. To optimize this problem, we can use the line search strategy or simplify the loss fuction to get an approximate result .

NOTE: For both GBDT classification and regression, we use the regression tree since we the residual is continuous.

Loss Function

Logistic loss —–for classification

$L(y, f(x)) = -\sum_{k=1}^K y_k \log p_k(x)$ , $p_k(x)$ is the probability of $x$ belongs to the $k$ class.

MeanSquare loss ——for regression

$L(y, f(x)) = (y-f(x))^2$
